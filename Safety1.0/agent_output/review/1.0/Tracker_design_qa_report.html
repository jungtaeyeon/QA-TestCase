#항목내용
Q1Status 값 재정의 기획서(Entry/QC/MA…)와 디자인(Converting/Entering…) Status 값이 전면 상이. 기획서 버전 구분 또는 디자인 근거 확인 필요.
Q2Source type vs Report Type 디자인 "Source type" 컬럼이 기획서 "Report Type"(C.1.3) 대체인지, 아니면 별도 필드인지 확인 필요. 기획서 2.5에 Source Type Classification이 별도 정의됨.
Q3Batch Receiver 출처 기획서 4.1 미포함 컬럼. 데이터 필드 매핑 및 기획서 추가 여부 확인 필요.
Q4Safety DB 캔버스 미반영 입력 URL(node-id=17339:390 Safety DB)에 Tracker 디자인 없음. 넵튠 [1.1] 기반 분석 수행. Safety DB 이관 계획 확인 필요.
