🎨 Figma 현황

드롭다운의 실제 옵션 목록 확인 불가

Figma에서 Serial Number Setting은 "Vertical Field" 인스턴스(드롭다운)로 구현되어 있으나, 드롭다운 펼침 상태의 옵션 목록이 디자인에 포함되어 있지 않아 정확한 범위 확인 불가.

기획서 자체에 본문(3~9)과 와이어프레임(5~9)이 불일치함. 디자인 적용 범위를 기획 측에 확인하고, 드롭다운 옵션 목록을 디자인에 명시 필요.
