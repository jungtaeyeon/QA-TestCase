|  | 내용 | 출처 |
| --- | --- | --- |
| 예시 A | `202604ST001` (일련번호 3자리) | Case ID.md › ICSR 식별자 관리 사양서 — Case vs Report Identifier 노출 사양 |
| 예시 B | `202604ST00002` (일련번호 5자리) | 동일 셀 내 두 번째 예시 |

**📌 기획 수정 필요 사항**:
