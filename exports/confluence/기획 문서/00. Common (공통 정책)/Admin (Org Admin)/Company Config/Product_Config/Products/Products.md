# 크리에이터블 드롭다운

## 설계 개요

* 대상 필드: Manufacturer(제조사) 등 마스터 데이터 관리가 필요한 필드
* 목적: 기존 리스트에서 빠른 검색 및 선택을 지원함과 동시에, 새로운 데이터를 즉시 등록하고 관리(삭제)할 수 있는 유연한 UI 제공

|  |  |  |
| --- | --- | --- |
| **구분** | **기능 상세 및 인터랙션** | **비고 및 제약 조건** |
| 검색 | 사용자가 필드에 타이핑을 시작하면 서버에 등록된 기존 리스트 중 매칭되는 값을 필터링하여 드롭다운에 표시 | 대소문자 구분 없음 |
| 항목 선택 | 드롭다운에 노출된 값을 클릭하면 해당 값이 필드에 자동으로 입력되며 드롭다운이 닫힘 |  |
| 신규 항목 추가 | 검색 결과가 없을 경우, 드롭다운 리스트 최하단에 [+ 추가하기] 버튼 노출 | 버튼 클릭 시 입력한 텍스트가 신규 리스트로 등록  데이터 입력 후 저장 버튼 클릭 필요 |
| 항목 관리 | 드롭다운 리스트의 각 항목에 마우스 호버 시 우측에 삭제(X) 아이콘이 노출 | 삭제 아이콘 클릭 시 해당 항목이 리스트에서 제거 |
| 데이터 동기화 | 신규 추가 또는 삭제된 리스트는 즉시 데이터화되어 서버 DB에 저장 | 이후 다른 제품 등록 시에도 업데이트된 리스트가 공통 적용 |
