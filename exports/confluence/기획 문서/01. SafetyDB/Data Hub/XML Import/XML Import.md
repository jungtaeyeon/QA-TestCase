## 2.7 필요 조건 적용 정책

### 2.7.1 Case ID Configuration을 진행하지 않은 경우

* Case ID Configuration에서 [조직자 이름]을 설정하지 않은 경우:

  + ‘Organisation Home에서 Case ID Configuration을 먼저 진행해주세요. Case ID 생성 규칙이 설정되지 않으면 케이스 생성이 불가합니다.’ 안내문구 팝업 표시.
  + 버튼은 ‘OK’하나.
