# AE Type 분류 (이상사례 구분 로직)

## 중대성 판정

* 우선순위 1.1 > 1.2 > 1.3 순으로 적용하며, 하나라도 TRUE이면 'Serious'로 판정한다.
* 사망 (E.i.3.2a), 생명의 위협(E.i.3.2b), 입원 또는 입원기간의 연장(E.i.3.2c), 중대한 불구나 기능저하(E.i.3.2d), 선천적 기형 초래(E.i.3.2e), 기타 의학적으로 중요한 상황(E.i.3.2f)

| 순위 | 조건 | 결과 |
| --- | --- | --- |
| 1.1 | 사망/치명적(E.i.3.2a-b) 중 하나라도 TRUE | 중대함 (사망/치명적) |
| 1.2 | 그 외 중대성(E.i.3.2c-f) 중 하나라도 TRUE | 중대함 |
| 1.3 | 전체(E.i.3.2a-f)가 Blank 또는 NI인 경우 | 중대하지 않음 |

## 인과성 판정 (Causality)

보수적 처리 원칙: 값 도출이 불가능할 경우, 안전을 위해 '인과성 있음'으로 간주한다.

| 항목 | 조건 | 결과 |
| --- | --- | --- |
| 자체 평가 | 평가 결과(자체 필드)가 Related | 인과성 있음 |
| KRCT | KRCT 평가 결과(G.k.9.i.2.r.3.KR.2)가 1 (Related) | 인과성 있음 |
| WHO-UMC | WHO-UMC 평가 결과(G.k.9.i.2.r.3.KR.1)가 1, 2, 3, 5, 6 중 하나 | 인과성 있음 |
| 누락 시 | 모든 필드에서 값을 확인할 수 없는 경우 | 인과성 있음 (보수적 처리) |

## 예측성 판정 (Expectedness)

자사 의약품에 대해서만 평가하며, 보수적 처리 원칙을 적용한다.

| 항목 | 조건 | 결과 |
| --- | --- | --- |
| 예상치 못함 | RSI 존재 여부(Expectedness/Listedness)가 False | 예상하지 못함 |
| 누락 시 | 값이 없는 경우(Blank) | 예상하지 못함 (보수적 처리) |

## Source Type별 최종 사례 분류

### **Group A. 시판 전 임상 단계**

(Source Type: b, c, i, 분류 불가) — 예측성(Expectedness)이 분류의 핵심

| AE Type | 조건 |
| --- | --- |
| SUSAR (사망/치명적) | 중대함(사망/치명적) + 인과성 있음 + 예상하지 못함 |
| SUSAR (기타) | 중대함 + 인과성 있음 + 예상하지 못함 |
| SADR | 중대함 + 인과성 있음 + 예상함 |
| SAE | 중대함 + 인과성 없음 |
| ADR | 중대하지 않음 + 인과성 있음 |
| AE | 중대하지 않음 + 인과성 없음 |

### **Group B. 시판 후 단계**

(Source Type: a, d, e, f, g, h, j, k) — 예측성과 관계없이 중대성과 인과성 위주로 분류

| AE Type | 조건 |
| --- | --- |
| SADR | 중대함 + 인과성 있음 |
| SAE | 중대함 + 인과성 없음 |
| ADR | 중대하지 않음 + 인과성 있음 |
| AE | 중대하지 않음 + 인과성 없음 |

## **최종 결과 데이터 (AE Type 정의)**

| AE Type | 정의 |
| --- | --- |
| SUSAR (사망/치명적) | 사망 또는 치명적인 예상하지 못한 중대한 약물이상반응 |
| SUSAR (기타) | 그 외 예상하지 못한 중대한 약물이상반응 |
| SADR | 중대한 약물이상반응 (Serious Adverse Drug Reaction) |
| SAE | 중대한 이상사례 (Serious Adverse Event) |
| ADR | 약물이상반응 (Adverse Drug Reaction) |
| AE | 이상사례 (Adverse Event) |
